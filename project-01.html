霓京 NEIKYO

一款以近未来都市、妖灵共存、抽卡契约与战斗养成为核心设定的幻想题材 UI 游戏界面设计。 整体视觉融合都市霓虹、神社意象、古典妖怪与赛博幻想，营造神秘、华丽且具有沉浸感的游戏体验。

Statement

作品说明

《霓京》是一个围绕抽卡、妖灵收集、角色养成与战斗出征展开的 UI 游戏界面设计项目。 作品尝试将近未来都市的冷光感与传统妖怪文化的神秘感结合，形成具有东方幻想色彩的游戏视觉系统。

World View

近未来都市
与妖灵契约

故事发生在近未来都市 「霓京」。 在这座城市中，人类与妖灵共存于表世界与里世界之间。 一种名为 「侵蚀」 的病毒正在扩散，使部分妖灵逐渐失控， 城市秩序也因此被打破。

玩家扮演 「调律者」，通过抽卡唤醒封印在古董卡牌中的古老妖灵， 与它们缔结契约并并肩作战。卡池中包含古典妖怪、赛博妖精与神代遗种。 都市霓虹与古老神社交错，玩家需要在霓虹雨夜中探索事件背后的真相。
